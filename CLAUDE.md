# CLAUDE.md — KDS Comfort Food

## Project

Catering website for Chef Yaya (Yajaira Springer). NYC-based, Hall of Fame chef.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Commands

```bash
npm run dev    # Development
npm run build  # Production build
npm run start  # Production server
```
